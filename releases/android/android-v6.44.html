Android v6.44

Official Release Notes

  • Now you can cancel attachment uploads while they're in progress. If a transfer is slowed because you're on the road (in airplane mode?) or the mediocre hotel WiFi is about to implode, you can wait to reload when more available bandwidth is bestowed.
  • We also fixed a bug that sometimes caused the “back” gesture to interrupt video trimming, and added notification profile support for Signal call events so that you can easily see when an incoming call was missed because a notification profile was enabled for that chat.

Upcoming Features
